
# Aurum Vendor Dashboard

Aurum Vendor Dashboard is a modern jewellery management web application built for vendors to manage products, orders, customers, and dashboard analytics efficiently.

## 🚀 Tech Stack

- React.js
- Vite
- JavaScript
- Tailwind CSS
