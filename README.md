
# Raymond Premium Tailoring

Raymond Premium Tailoring is a luxury tailoring showcase platform designed to highlight bespoke fashion creations and custom tailoring expertise.
It provides an elegant dashboard for managing portfolio posts, drafts, and customer inquiries with a seamless user experience.

## 🚀 Tech Stack

- React.js
- Vite
- JavaScript
- Tailwind CSS
